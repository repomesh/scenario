# Noise sample licenses

These bundled noise samples are synthesised procedurally at build time and
dedicated to the public domain under [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/).

Replacing them with real-world recordings is possible: drop CC0-licensed
WAV files (24kHz mono, PCM16) at the matching filenames. Sources:

- cafe.wav — background chatter with ambient noise
- street.wav — traffic, distant sirens
- office.wav — keyboards, HVAC hum
- airport.wav — announcements, crowd murmur
- babble.wav — speech babble for the `multiple_voices` effect (NOT a
  `background_noise` preset; see proposal §4.5 L521 vs L533).

If you replace a sample with copyrighted audio, add attribution and
license details here.
