# `GeminiLiveAgentAdapter`

Connects to Google's Gemini Live API over WebSocket. Native VAD and barge-in
are built-in.

## Constructor

```python
import os
import scenario

adapter = scenario.GeminiLiveAgentAdapter(
    model="gemini-2.5-flash-native-audio-latest",
    voice="Algieba",
    system_instruction="You are a helpful assistant.",
    api_key=os.environ.get("GEMINI_API_KEY"),
)
```

If `api_key` is not passed, it falls back to `GEMINI_API_KEY` in the
environment.
