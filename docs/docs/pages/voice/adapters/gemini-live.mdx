# `GeminiLiveAgentAdapter`

Connects to Google's Gemini Live API over WebSocket. Native VAD and barge-in
are built-in.

## Constructor

:::code-group

```python [python]
import os
import scenario

adapter = scenario.GeminiLiveAgentAdapter(
    model="gemini-2.5-flash-native-audio-latest",
    voice="Algieba",
    system_instruction="You are a helpful assistant.",
    api_key=os.environ.get("GEMINI_API_KEY"),
)
```

```typescript [typescript]
import scenario, { voice } from "@langwatch/scenario";

const { GEMINI_LIVE_MODEL } = voice;

const adapter = scenario.geminiLiveAgent({
  model: GEMINI_LIVE_MODEL, // "gemini-2.5-flash-native-audio-latest"
  voice: "Algieba",
  systemInstruction: "You are a helpful assistant.",
  // apiKey falls back to GEMINI_API_KEY / GOOGLE_API_KEY
});
```

:::

If `api_key` / `apiKey` is not passed, it falls back to `GEMINI_API_KEY` (and, in
TypeScript, `GOOGLE_API_KEY`) in the environment.

<Callout type="info">
  In TypeScript, the Gemini Live adapter uses `@google/genai`, declared as an
  **optional peer dependency** — install it yourself
  (`pnpm install @google/genai`) if you import this adapter.
</Callout>

## Capabilities

| streaming_transcripts | native_vad | dtmf | interruption | input_formats | output_formats |
| --- | --- | --- | --- | --- | --- |
| ✅ | ✅ | ❌ | ✅ | `pcm16/16000` | `pcm16/24000` |

<Callout type="info">
  The **TypeScript** `geminiLiveAgent` declares `interruption: true` (it cuts
  its in-flight audio on the next activity-start marker). The Python capability
  matrix currently lists Gemini interruption as deferred — see the
  [capability matrix](/voice/capability-matrix) for the per-SDK values.
</Callout>
