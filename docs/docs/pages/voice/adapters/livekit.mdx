# `LiveKitAgentAdapter` (preview)

Designed to join a LiveKit room as a participant and drive a voice agent over
WebRTC.

<Callout type="info">
  **TypeScript: not yet available.** `LiveKitAgentAdapter` is Python-only today —
  it is not exported from `@langwatch/scenario` (tracked in
  [#563](https://github.com/langwatch/scenario/issues/563)).
</Callout>

<Callout type="warning">
  **Transport not yet implemented.** Calling `connect()` raises
  `PendingTransportError`. The capability declaration is final (it matches the
  wire spec); only the transport glue code is pending. Track progress in
  follow-up issues on the [scenario repo](https://github.com/langwatch/scenario/issues).
</Callout>

## Constructor

```python
import os
import scenario

adapter = scenario.LiveKitAgentAdapter(
    url=os.environ["LIVEKIT_URL"],
    api_key=os.environ["LIVEKIT_API_KEY"],
    api_secret=os.environ["LIVEKIT_API_SECRET"],
    room="my-room",
)
```
